O que é a Soma?

A soma (ou adição) é uma das quatro operações fundamentais da aritmética. Consiste em combinar dois ou mais números para obter um total, chamado de soma ou resultado.

💡 Definição formal: A adição é a operação que associa a cada par de números a e b um terceiro número c, de modo que a + b = c.

Propriedades da Adição

1. Comutatividade

A ordem das parcelas não altera o resultado:

a + b = b + a
Exemplo

3 + 5 = 8  e  5 + 3 = 8

2. Associatividade

A forma de agrupar as parcelas não altera o resultado:

(a + b) + c = a + (b + c)
Exemplo

(2 + 3) + 4 = 9  e  2 + (3 + 4) = 9

3. Elemento Neutro

O zero é o elemento neutro da adição. Somar zero a qualquer número não altera seu valor:

a + 0 = a

4. Fechamento

A soma de dois números naturais é sempre um número natural — a operação é "fechada" dentro do conjunto:

a ∈ ℕ, b ∈ ℕ ⟹ a + b ∈ ℕ

O que é a Subtração?

A subtração é a operação inversa da adição. Consiste em retirar uma quantidade de outra, obtendo a diferença entre os dois números.

💡 Na expressão a − b = c, a é o minuendo, b é o subtraendo e c é a diferença.

Exemplo Prático

Exemplo

Se você tem 15 maçãs e dá 7 para um amigo:

15 − 7 = 8

Você fica com 8 maçãs.

Propriedades Importantes

Não é comutativa

Diferente da adição, a ordem importa na subtração:

Exemplo

10 − 3 = 7  mas  3 − 10 = −7

Elemento Neutro

Subtrair zero de qualquer número não altera seu valor:

a − 0 = a
⚠️ Atenção

A subtração não é associativa! Veja: (10 − 5) − 2 = 3, mas 10 − (5 − 2) = 7.

Relação com a Adição

Toda subtração pode ser verificada pela adição correspondente:

a − b = c ⟺ b + c = a

O que é a Multiplicação?

A multiplicação é uma forma abreviada de realizar somas repetidas. Multiplicar a por b equivale a somar a consigo mesmo b vezes.

a × b = a + a + a + ...  (b vezes)
Exemplo

4 × 3 significa somar o 4 três vezes:

4 + 4 + 4 = 12

Propriedades

1. Comutatividade

a × b = b × a

2. Associatividade

(a × b) × c = a × (b × c)

3. Elemento Neutro

O número 1 é o elemento neutro da multiplicação:

a × 1 = a

4. Elemento Absorvente

Qualquer número multiplicado por zero resulta em zero:

a × 0 = 0

5. Distributividade

A multiplicação distribui sobre a adição:

a × (b + c) = (a × b) + (a × c)
Exemplo

3 × (4 + 5) = 3 × 4 + 3 × 5 = 12 + 15 = 27

O que é a Divisão?

A divisão é a operação inversa da multiplicação. Consiste em repartir uma quantidade em partes iguais.

💡 Na expressão a ÷ b = c, a é o dividendo, b é o divisor e c é o quociente.

a ÷ b = c  ⟺  b × c = a

Divisão Exata e Não Exata

Divisão Exata

Quando o resto é zero:

Exemplo

20 ÷ 4 = 5  (resto 0)

Divisão com Resto

Quando o resto é diferente de zero. O resultado completo é expresso como:

dividendo = (divisor × quociente) + resto
Exemplo

21 ÷ 4 = 5  com resto  1

Verificação: 4 × 5 + 1 = 21 ✓

⚠️ Importante

Não é possível dividir por zero! A divisão por zero é indefinida na matemática.

O que são Frações?

Uma fração representa uma ou mais partes de um todo que foi dividido em partes iguais. É escrita na forma a/b, onde a é o numerador e b é o denominador.

💡 O denominador indica em quantas partes o todo foi dividido; o numerador indica quantas dessas partes estamos considerando.

Tipos de Frações

1. Fração Própria

O numerador é menor que o denominador (valor < 1):

Exemplos

1/2,   3/4,   2/5

2. Fração Imprópria

O numerador é maior ou igual ao denominador (valor ≥ 1):

Exemplos

5/3,   7/4,   9/9

3. Número Misto

Combinação de um número inteiro com uma fração própria:

Exemplo

5/3 = 1 + 2/3 = 1⅔  (um inteiro e dois terços)

Operações com Frações

Soma com denominadores iguais

a/b + c/b = (a + c) / b

Soma com denominadores diferentes

Encontre o MMC ou use a fórmula direta:

a/b + c/d = (a×d + c×b) / (b×d)
Exemplo

1/3 + 1/4 = (1×4 + 1×3) / (3×4) = 7/12

Multiplicação de frações

(a/b) × (c/d) = (a×c) / (b×d)

Divisão de frações

Multiplique pela fração inversa:

(a/b) ÷ (c/d) = (a/b) × (d/c) = (a×d) / (b×c)

O que é Potenciação?

A potenciação é uma forma abreviada de representar multiplicações repetidas de um mesmo número. Escreve-se aⁿ, onde a é a base e n é o expoente.

aⁿ = a × a × a × ... × a  (n vezes)
Exemplo

2⁴ = 2 × 2 × 2 × 2 = 16

Propriedades da Potenciação

1. Produto de potências de mesma base

Mantém a base e soma os expoentes:

aᵐ × aⁿ = aᵐ⁺ⁿ

2. Quociente de potências de mesma base

Mantém a base e subtrai os expoentes:

aᵐ ÷ aⁿ = aᵐ⁻ⁿ

3. Potência de potência

Mantém a base e multiplica os expoentes:

(aᵐ)ⁿ = aᵐˣⁿ

4. Expoente zero

Qualquer número (exceto zero) elevado a zero é igual a 1:

a⁰ = 1  (a ≠ 0)

5. Expoente negativo

Resulta no inverso da potência com expoente positivo:

a⁻ⁿ = 1 / aⁿ
Exemplo

2⁻³ = 1 / 2³ = 1 / 8 = 0,125

⚠️ Atenção

0⁰ é uma forma indeterminada — seu valor depende do contexto matemático.
